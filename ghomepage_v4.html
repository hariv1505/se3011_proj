MSM Module - The Gs

Available Releases:









Version 1.2
Download Date of Release: 1/05/2014

Compatibility:
Windows XP, Vista, 7, 8, 8.1
Linux Ubuntu 12, 13


UPDATE:
Amendments have been made to accomodate for various inputs, whilst accounting for user error. README.txt has also been provided.
Please use:
<command to run> -h
on the command line, or read README.txt.

This version, reads CSV files and then analyses, using the Momentum strategy, makes choices to buy and sell stocks according to the data provided. It then produces a resultant file with updated trades. The module requires two parameters; The required CSV file to be read in and the parameters file - both which are provided in the module

Download the zip file, then extract it to a destination folder of your choice. To use the module in standalone mode, using the Windows Command Prompt, change directory to the newly unzipped directory. When navigated into the directory, run msm_v_1_2.exe using the command:

msm_v_1_2.exe
for Windows, or

wine msm_v_1_2.exe
for Linux

Alternatively, the application can be double-clicked.

The application can also be called from other programs. An example of doing so is using the Process class, in Java. The namely example is:

Process p = Runtime.getRuntime().exec("cmd /C /msm_v_1_2");

which will generate an output that can be read in using the BufferedReader class

To change the parameters for the file, namely the 'n' (window size for Simple Moving Averages) and 'threshold' values, please open 'parameters.param' using a text-editor. You will notice that the first number represents 'n' whereas the second number represents the threshold decimal.

The required files are in the directory, and after execution the resultant CSV and log file will be created in the directory.
